# EraseUnaVez
Desarrollar y lanzar una aplicación móvil que combine cuentos clásicos con tecnología de generación de historias por inteligencia artificial (Gemini) para aumentar el interés y la participación creativa en la lectura infantil en un entorno digital seguro y accesible.

Este proyecto está dividido en tres componentes principales que interactúan entre sí:

| Componente | Tecnología Principal | Propósito | Documentación Detallada |

| :--- | :--- | :--- | :--- |
| **Frontend** | **React Native** (con Expo Go) | Interfaz de usuario y aplicación móvil. | [Ver README del Frontend] |
| **Backend** | **Node.js** (con Express/NestJS, etc.) | Lógica del negocio, manejo de API y conexión con Gemini. | [Ver README del Backend]|
| **Base de Datos** | **MySQL** | Almacenamiento persistente de datos (usuarios, historias, etc.). | - |
